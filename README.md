# Warframe Market Terminal

## Setup

```bash
bun install
```

## Development

```bash
bun run dev
```

## Production

```bash
bun run build
bun start
```

* Or to deploy using pm2:
```bash
bun run deploy
```

## Tuning (optional)

```bash
cp config.yaml.example config.yaml
cp .env.local.example .env.local
```
